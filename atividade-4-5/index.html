Atividade 4

Entrega: 26/04/2021

Nesta atividade, você deverá mostrar que consegue posicionar elementos na tela com:

  • display: block, inline e inline-block
  • float: left, right
  • clear: left, right, both

Crie uma página web com quatro links de navegação:

  1. Página do Integrante 1
  2. Página do Integrante 2
  3. Link para o trabalho anterior
  4. Link para a página da Telecom

As páginas dos integrantes podem compartilhar o mesmo CSS, mudando apenas o conteúdo do HTML.

O formato de entrega será um vídeo apresentando no Browser quais elementos são inline-block, além do que foi feito para posicionar os elementos. Mostre de forma específica qual a estratégia para posicionamento dos elementos usando display e float.

Note que não precisa seguir o que foi apresentado em aula religiosamente, tente reposicionar os elementos HTML de uma forma mais criativa.

Atividade 5

Entrega: 03/05/2021

Nesta atividade, você deve mostrar que entende os seguintes itens:

  1. Uso de estados em CSS
  2. Uso de transições

Para demonstrar conhecimento, você deverá definir as propriedades de transições de algum marcador presente no HTML e fazer a mudança de estado em algum evento, como o ato de passar o mouse por cima do marcador.

Como formato de entrega, você deverá submeter um vídeo mostrando que a sua página possui os itens mencionados acima.
